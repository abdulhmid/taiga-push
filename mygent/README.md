# Great Agent Skill Collection

Koleksi skill agent berkualitas tinggi untuk pengembangan **Python FastAPI Backend** dan **Vue.js Dashboard**.

## Skills yang Tersedia

| Skill Name                        | Description                                      | Tags                     |
|-----------------------------------|--------------------------------------------------|--------------------------|
| python-api-fastapi                | Membuat RESTful API production-ready dengan FastAPI | python, fastapi, backend |
| vuejs-dashboard                   | Membuat modern dashboard/admin panel dengan Vue 3 | vue, vue3, dashboard     |

## Cara Menggunakan

### Untuk Agent (Claude, Cursor, Copilot, dll.)
1. Clone repo ini atau tambahkan sebagai submodule.
2. Load `constitution.md` sebagai system prompt utama.
3. Agent akan otomatis memilih skill dari folder `skills/` berdasarkan deskripsi di SKILL.md.

### Untuk Pengembangan
```bash
git clone https://github.com/abdulhmid/great-agent-skill.git
cd great-agent-skill
